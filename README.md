OSDC watches security patches landing in open source every day. It extracts vulnerability patterns, compares them to its historical database, and flags recurring antipatterns across languages and ecosystems.

## How it works

Three automated runs per day (06:00, 14:00, 23:00 UTC) pull new advisories from the GitHub Advisory Database, extract and filter the fix diffs, analyze them with an LLM, and match the vulnerability pattern against a growing historical database. Each pattern gets a normalized ID that enables cross-language, cross-ecosystem correlation.
