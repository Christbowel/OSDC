## GHSA-r95r-rj6r-c39x · @earendil-works/pi-coding-agent

2026-06-17 · JavaScript · **LOW** 2.2 · Pattern: `RACE_CONDITION→TOCTOU`

### Root cause

The vulnerability stemmed from a race condition during the writing of the authentication file. The `writeFileSync` operation, which writes the file content, and the subsequent `chmodSync` operation, which sets file permissions, were not atomic. ### Impact

An attacker could potentially exploit this race condition to read sensitive credentials stored in the `auth.json` file if they could access the file system during the vulnerable window, leading to unauthorized access.
